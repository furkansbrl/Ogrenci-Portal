<h1>Öğrenci Portalı MVC</h1>

#Öğrenci kaydı oluşturup kayıtları tutan ve yöneten MVC (Model-View-Controller) mimarisine sahip temel bir uygulamadır.

## Bağımlılıklar
> [!IMPORTANT]
> .NET 8 SDK
> 
> EntityFrameCore.SqlServer
> 
> EntityFrameCore.Tools
